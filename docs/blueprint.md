# **App Name**: EmotiCare

## Core Features:

- Consent-Based Authentication: User authentication and consent management for data collection (webcam, audio, text).
- Multimodal Check-in: Real-time multimodal check-in (face, voice, text) to capture user's emotional state.
- Mood Prediction and Action Suggestion: AI-powered tool that fuses multimodal data to predict mood and suggest actions.
- Digital Journal: Journaling feature with text editor and sentiment analysis of entries.
- AI Chat Companion: AI chat companion that provides CBT-informed support and journaling prompts.
- Mood Timeline: Mood timeline visualization to track emotional trends over time.
- Emergency Support: Emergency contact and helpline integration for users in crisis.

## Style Guidelines:

- Primary color: Gentle lavender (#E6E6FA) to promote calm and reflection.
- Background color: Very light gray (#F5F5F5) to provide a clean and unobtrusive backdrop.
- Accent color: Soft sky blue (#B0E2FF) for interactive elements and highlights, creating a sense of ease.
- Body and headline font: 'PT Sans' (sans-serif) to provide a modern look while retaining warmth and personality, making it accessible and readable.
- Use soft, rounded icons representing emotions and actions.
- Clean and intuitive layout with clear content hierarchy to minimize user overwhelm.
- Subtle, calming animations to provide gentle feedback and enhance user experience.
